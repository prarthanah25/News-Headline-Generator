# News Headline Generator Using Machine Learning

This project uses Transformer-based NLP models to generate short headlines from long news articles.

## Technologies Used
- Python
- Streamlit
- HuggingFace Transformers
- NLP
- BART Model

## Features
- Automatic headline generation
- News summarization
- Interactive frontend
- Data preprocessing and visualization

## Model Used
- sshleifer/distilbart-cnn-12-6

## Run Project

streamlit run app.py
